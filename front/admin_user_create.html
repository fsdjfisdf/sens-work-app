ADMIN ONLY

사용자 계정 관리

관리자만 계정을 만들고, 비밀번호를 초기화하고, 퇴사자 계정을 비활성화할 수 있습니다.

CREATE ACCOUNT

신규 계정 생성

여기서 입력하는 이름은 DB의 nickname 컬럼에 저장됩니다.

A Active

정상 사용 상태입니다. 로그인 가능합니다.

I Inactive

비활성화 상태입니다. 퇴사자, 휴직자, 사용 중지 계정에 사용합니다. 로그인 불가입니다.

D Deleted

삭제 처리 상태입니다. 일반 운영에서는 보통 직접 쓰지 않고, 내부 정리용 상태로 두는 편이 안전합니다.

영문으로 시작하는 6~20자 영문/숫자
DB의 nickname 컬럼에 저장됩니다.

PASSWORD RESET

관리자 비밀번호 초기화

대상 사용자의 userIdx를 기준으로 초기 비밀번호를 다시 설정합니다.

USER LIST

현재 사용자 목록

퇴사자는 비활성화(I) 처리하면 로그인할 수 없습니다. 복직이나 재사용 시 다시 활성화(A)하면 됩니다.
